# git

### 运行前的配置：

#### $ git config --global user.name "John Doe"



#### $ git config --global user.email johndoe@example.com



### 检查配置信息:


####$ git config --list

### 初始化仓库：

#### $ git init



### 克隆现有的仓库

#### $ git clone https://github.com/libgit2/libgit2 mylibgit##自定义仓库名字



### 记录每次更新到仓库

#### 工作目录下的每一个文件都不外乎这两种状态：已跟踪或未跟踪
